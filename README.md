## Installation et exécution de l'application "Meetings"
1. Lancez mongoDB : Sous Windows, repertoire C:\Program Files\MongoDB\Server\3.4\bin, commande `mongod` sous invite de commande.
2. Se rendre de le repertoire de l'application, exécutez : `npm i`
3. Exécuter `node meetings`
4. Ouvrez l'application depuis la page `http://localhost:8080`
